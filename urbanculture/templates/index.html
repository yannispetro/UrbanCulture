Explore Urban Landscapes

Urban Culture is a neighborhood recommendation system that matches geographical areas with your interests. Select one out of the 80 available cities, express your preferences via a few keywords and start exploring!

The system is built using machine learning and natural language processing (NLP) techniques. A fastText word embedding was learned on Airbnb and Yelp data and several scikit-learn tools are utilized for geospatial clustering and semantic similarity estimation. The website is built using Flask and Bootstrap and deployed on Heroku.
